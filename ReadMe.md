## 编译器优化理解

请尝试用-O2或-O3对程序进行优化，可进行常量的合并与传播、公共表达式消减、函数内联、尾递归消除等。大家可尝试编写例子尝试。

学会使用宏指令或openmp发挥CPU的向量运算以及多核并行技术来提高程序的性能。

新型的GPU硬件都有很高的浮点并行运算能力，如英伟达提供的CUDA，目前主要用于人工智能。大家有必要可尝试安装CUDA进行测试。
